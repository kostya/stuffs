# stuffs

Some stuffs which i used in every project for Crystal. Mini ActiveSupport.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  stuffs:
    github: kostya/stuffs
```


## Usage


```crystal
require "stuffs"
```
